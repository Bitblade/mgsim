bool Simulator::drisc::RegisterFile::WriteRegister ( const RegAddr &  addr,
const RegValue &  data,
bool  from_memory 
)

Writes a register

Writing a register may in some cases result in the data in the target register being placed in the data parameter. This happens when a waiting state is written back to a register which is full. The resulting state of the parameter will be full, such that the caller can reschedule the thread.

Parameters
[in]addrthe address of the register to write
[in]datathe data to write to the register.
[in]from_memoryindicates if the write comes from memory.
[in]sourcethe source of the write. This is used for arbitration in certain cases.
Returns
true if the register could be written

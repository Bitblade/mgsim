LFID  fid,
ContextType  context,

)

Allocates registers.

Parameters
size[in]Number of registers to allocate for each register type.
fid[in]Family that makes the request. For debugging purposes only.
reserved[in]Whether we're allocating a reserved context at least.
indices[out]Array that will receive the base indices of the allocated registers.
Returns
false if not enough register were available

A context can be reserved with ReserveContext. Allocating registers with reserved false will only allocate the registers if all reserved contexts can remain available. If reserved is true, at least one reserved context worth of register is considered for allocation as well.

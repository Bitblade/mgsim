4 === Overview of DDR (Double Data Rate) SDRAM ===
5 
6 DDR uses modules ("DIMMs") that each has a number of DRAM devices ("chips")
7 on them. The number of devices on a module is a multiple of 8 (Non-ECC) or
8 9 (ECC). The devices on a module can be grouped into 1, 2 or 4 ranks.
9 Only 1 rank on a module can be active. Each device has a 4 or 8 bit data path.
10 Thus, for the full 64-bit data path, 16 or 8 devices are accessed in parallel.
11 
12 DDR has a prefetch buffer that buffers N consecutive words in a single read
13 and streams them out on the I/O bus, running at a higher frequency (a prefetch
14 buffer of 8 words requires a frequency multiplier on a double-date rate I/O
15 bus of 4).
16 
17 For instance, the Micron MT18JSF25672PD is a 2 GB DDR3 DIMM with a 64-bit
18 data bus and ECC. It uses 18 (sixteen + two for ECC) Micron MT41J128M8 memory
19 devices, which have a 128 MB capacity and a 8-bit data bus, arranged in 2
20 ranks.
21 The 128 MB memory device internally has eight banks, each with 16384 rows of
22 128 columns of 64 bits. Per memory clock cycle, 64 bits can be read with
23 pipelined column reads. These 64 bits are latched and returned over an 8-bit
24 databus in 4 double pumped I/O bus cycles (which has a 4x multiplier w.r.t.
25 the memory clock).
26 
27 Accessing a bank in a memory device involves the following:
28 * If the wrong row is opened, it must be precharged (closed) with a Row
29  Precharge (RP). This has a latency of tRP cycles.
30 * Opening the desired row. It takes tRCD (RAS to CAS Delay cycles before the
31  columns in the row can be read. Multiple columns can be read or written
32  without reopening the row.
33 * Selecting the column from an open row. The data is available on the output
34  pins after a tCL (CAS Latency) latency.
35 
36 As such, the latency for a read is:
37 * tCL + tRCD cycles if no row is open.
38 * tCL + tRCD + tRP cycles if the wrong row is open
39 * tCL cycles if the correct row is open.
40 
41 Note that a row cannot be precharged (closed) until at least tWR (Write
42 Recovery) cycles after the last write and at least tRAS (Row Active to
43 Precharge Delay) cycles after opening the row.
44 
45 Each generation of DDR increases the maximum size of the memory device and
46 doubled the I/O bus frequency multiplier (and, consequently, the prefetch
47 buffer size). Below is an overview of some common DDR modules:
48

51 DDR1-200 100 Mhz 100 MHz 1.6 GB/s PC-1600
52 DDR1-266 133 MHz 133 MHz 2.1 GB/s PC-2100
53 DDR1-333 166 MHz 166 MHz 2.7 GB/s PC-2700
54 DDR1-400 200 MHz 200 MHz 3.2 GB/s PC-3200
55 DDR2-400 100 Mhz 200 MHz 3.2 GB/s PC2-3200
56 DDR2-533 133 MHz 266 MHz 4.2 GB/s PC2-4200

62 DDR3-1333 166 MHz 667 MHz 10.6 GB/s PC3-10600
63 DDR3-1600 200 MHz 800 MHz 12.8 GB/s PC3-12800
64 
65 Bandwidth of DDR memory is identified by the number behind it. For instance,
66 DDR3-800 means the data rate is 800 MHz (thus, the I/O bus frequency is 400
67 MHz and the memory clock 100 MHz). Together with a 64-bit wide databus and
68 2 transfers/cycle, DDR3-800 can support up to 6.4 GB/s.
69 */
